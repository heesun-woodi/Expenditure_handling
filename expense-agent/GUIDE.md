# 개인카드 지출결의서 자동 작성 가이드

> Slack에서 영수증 사진을 올리면 AI가 자동으로 지출결의서를 만들어드립니다.

---

## 빠른 시작

0. 영수증을 올릴 Slack 채널에 **Mr.Simpson** 봇을 초대
1. 프로젝트 Slack 채널에서 `@Mr.Simpson` 멘션 + 영수증 사진 첨부
2. 지출결의서 자동 생성 (1~2분 소요)
3. 내용 확인 후 수정 → **제출** 버튼 클릭
4. 은미님 검토 → 입금 완료

---

## 0단계: 사전 준비 (처음 한 번만)

영수증을 올릴 Slack 채널에 **Mr.Simpson** 봇을 초대해야 합니다.

1. 해당 채널에서 `/invite @Mr.Simpson` 입력 후 전송
2. 봇이 채널에 참여했다는 메시지 확인

> **채널 조건:** 반드시 해당 프로젝트의 **던전이 있는 채널**이어야 합니다.
> 퀘봇(`/quest`)을 사용하는 채널과 동일한 채널을 사용하세요.
> 던전이 없는 채널에서 사용하면 프로젝트명을 자동으로 가져올 수 없습니다.

---

## 1단계: 영수증 업로드

프로젝트 채널에서 아래와 같이 메시지를 보냅니다.

```
@Mr.Simpson (영수증 사진 첨부)
```

**지원 형식:** JPG, PNG, HEIC, PDF
**한 번에 최대:** 10장

> **여러 장 한꺼번에 가능합니다.**
> 같은 날 여러 건을 썼다면 사진을 모두 한 번에 첨부하세요.

### 주의사항
- 반드시 **던전이 있는 프로젝트 채널**에서 사용해야 합니다. (DM이나 다른 채널 불가)
- 영수증 없이 멘션만 하면 안내 메시지가 옵니다.
- 영수증 사진은 **선명하게** 찍어주세요. 흐리면 금액/날짜를 잘못 읽을 수 있습니다.

---

## 2단계: 지출결의서 검토

잠시 후 봇이 스레드에 답장합니다.

```
Joseph님, 지출결의서가 작성되었습니다.
내용을 검토하고 수정이 필요하면 수정을 하신 후에 '제출' 버튼을 눌러주세요.

📄 https://docs.google.com/spreadsheets/d/...

---
📌 확인 사항:
- 날짜가 정확한가요?
- 금액이 맞나요?
- 항목 분류가 적절한가요?
- 목적 설명이 적절한가요?

[ 제출 ]
```

링크를 클릭해서 **Google Sheets에서 직접 수정**할 수 있습니다.

### 자주 수정하는 항목
| 항목 | 위치 | 설명 |
|------|------|------|
| 날짜 | F열 | AI가 잘못 읽었을 경우 직접 수정 |
| 금액 | O열 | 영수증 총액 |
| 항목 분류 | H열 | 드롭다운에서 선택 가능 |
| 목적 | J열 | 사용 목적 설명 |

---

## 3단계: 제출

수정이 완료되면 스레드의 **제출** 버튼을 클릭합니다.

> 제출은 **본인만** 가능합니다. 다른 사람이 누르면 오류 메시지가 옵니다.

제출 완료 시 재무팀 채널로 자동 전달됩니다.

---

## 4단계: 이후 프로세스 (자동)

제출 이후는 모두 자동으로 처리됩니다.

```
제출 완료
   ↓
은미님 검토 (Google Sheets 확인)
   ↓
은미님이 ✅ 이모지 클릭
   ↓
내 채널 스레드에 "확인하셨습니다" 알림
   ↓
입금 처리 후 "입금완료" 알림
```

---

## 자주 묻는 질문

**Q. 여러 달에 걸친 영수증이 있어요.**
→ 한꺼번에 올려도 됩니다. 파일 제목에 자동으로 월이 모두 표시됩니다.
예: `푸드케어 CRO PJ_26년 2월 3월_개인카드지출결의서_Joseph`

**Q. AI가 금액을 잘못 읽었어요.**
→ Google Sheets에서 직접 수정 후 제출 버튼을 누르세요.

**Q. 영수증 분석에 실패했다는 경고가 떴어요.**
→ 해당 영수증은 시트에 누락된 상태입니다. Google Sheets에서 직접 행을 추가하거나, 해당 영수증만 다시 올려주세요.

**Q. 제출 버튼이 안 눌려요.**
→ 본인이 올린 영수증만 제출 가능합니다. 본인 계정으로 올렸는지 확인해주세요.

**Q. 실수로 잘못 제출했어요.**
→ 재무팀 채널에서 은미님에게 직접 말씀해주세요.
