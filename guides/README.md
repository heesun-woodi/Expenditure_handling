# Mr.Simpson 슬랙봇 사용 가이드

> Mr.Simpson은 업무 자동화를 도와주는 슬랙봇입니다.

---

## 기능 목록

| 기능 | 설명 | 가이드 |
|------|------|--------|
| 지출결의서 자동 작성 | 영수증 사진을 올리면 AI가 지출결의서를 자동 생성 | [가이드 보기](./expense-report-guide.md) |
| 이메일 주소 추출 | 이모지 반응한 사람들의 이메일을 자동 추출 | [가이드 보기](./email-extractor-guide.md) |

---

## 공통 사전 준비

Mr.Simpson 봇을 사용하려면, 해당 Slack 채널에 봇이 초대되어 있어야 합니다.

1. 채널에서 `/invite @Mr.Simpson` 입력
2. 봇이 채널에 참여했다는 메시지 확인

> 새로운 기능이 추가되면 이 문서에 업데이트됩니다.
