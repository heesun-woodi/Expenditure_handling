# 이메일 주소 추출 가이드

> Slack 메시지에 이모지를 누른 사람들의 이메일 주소를 자동으로 추출합니다.

---

## 이런 때 사용하세요

- 공유회/세미나 참석자에게 캘린더 초대를 보내야 할 때
- 투표 결과에서 특정 이모지를 선택한 사람들에게 연락해야 할 때
- 이모지 반응한 사람들의 이메일 목록이 필요할 때

---

## 사용 방법

### 1단계: 편지 이모지 반응

이메일을 추출하고 싶은 메시지에 ✉️ (편지) 이모지를 달아주세요.

`[스크린샷: 메시지에 ✉️ 편지 이모지를 다는 화면]`

---

### 2단계: 추출할 이모지 선택

Mr.Simpson이 해당 메시지의 스레드에 답장합니다.

```
✉️ 이메일 이모지를 누르셔서 달려왔습니다.
이메일을 추출해 드릴까요?
어떤 이모지를 누른 분들의 이메일을 추출할까요?

[ ❤️ (11명) ] [ 4️⃣ (5명) ] [ 1️⃣ (11명) ] [ 2️⃣ (11명) ] [ 3️⃣ (6명) ] [ 전체 (22명) ]
```

원하는 이모지 버튼을 클릭하세요.
- **개별 이모지**: 해당 이모지를 누른 사람들의 이메일만 추출
- **전체**: 모든 이모지를 누른 사람들의 이메일을 추출 (중복 제거)

`[스크린샷: Mr.Simpson이 이모지 선택 버튼을 보여주는 스레드 메시지]`

---

### 3단계: 이메일 명단 수신

Mr.Simpson이 **DM**으로 이메일 명단을 보내줍니다.

```
📋 요청하신 이메일 명단이 준비되었습니다.

Elan@mfitlab.com,
david@mfitlab.com,
lucid.h.park@gmail.com,
nicky@mfitlab.com,
paul@mfitlab.com,

💌 From your devoted concierge, Mr. Simpson
```

`[스크린샷: DM으로 수신한 이메일 명단]`

---

### 4단계: 활용

이메일 명단을 복사하여 원하는 곳에 붙여넣기하세요.

**활용 예시:**
- **구글 캘린더 초대**: 이메일 복사 → 캘린더 이벤트 생성 → 참석자 추가에 붙여넣기
- **이메일 발송**: 수신자 목록에 붙여넣기
- **문서 공유**: Google Drive 공유 대상에 붙여넣기

`[스크린샷: 구글 캘린더에서 참석자 추가란에 이메일을 붙여넣는 화면]`

---

## 자주 묻는 질문

**Q. 편지 이모지를 눌렀는데 봇이 반응하지 않아요.**
→ 해당 채널에 Mr.Simpson 봇이 초대되어 있는지 확인하세요. (`/invite @Mr.Simpson`)

**Q. 추가로 다른 이모지의 이메일도 추출하고 싶어요.**
→ 같은 메시지에 ✉️ 편지 이모지를 다시 눌러주세요.

**Q. 외부인(게스트)의 이메일도 추출되나요?**
→ Slack에 등록된 이메일 기준으로 추출됩니다. 게스트 계정도 Slack에 등록된 이메일이 추출됩니다.

**Q. 이메일이 쉼표로 구분되어 있는 이유가 뭔가요?**
→ 구글 캘린더, Gmail 등에 바로 붙여넣기할 수 있도록 쉼표로 구분되어 있습니다.
